## Comandos:
#### Inicialização do Projeto:
```
npm init
```
```
npm init -y
```

---

#### Node Package Manager (NPM):
- O Comando acima cria um arquivo de <u>configuração do Projeto</u>: << package.json >>
<br>
- Serve para gerenciar os Pacotes (bibliotecas, funcionalidades e outros recursos externos, de terceiros) necessários para o funcionamento do projeto.


---
#### Criando Scripts personalizados no arquivo << package.json >>:
Exemplos de funcionalidades básicas para execução de scripts:
- Iniciar o servidor.
- Build (compilar algum arquivo).
- Teste.

```
"scripts": {
    "start": "npm ./main.js",
}
```
Obs.1: Para rodar os Scripts, temos os seguintes comandos "facilitadores" (que rodam o código conforme o descrito):
```
npm start
```
```
npm run teste
npm run ...
```
